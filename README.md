Snowdrop
========

No-fuss CSS+HTML boilerplate for people who *like* CSS and HTML.

No preprocessors, no build step, not a lot of naming hoohaw, no fuss. 

[snowdrop.axoplasm.com](//snowdrop.axoplasm.com)
